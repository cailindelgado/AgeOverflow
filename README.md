# CSSE6400 Cloud Infrastructure Assignment
## AgeOverflow

Demonstrate your ability to _design_, _implement_, and _deploy_ a web API that can process a high load,
i.e. a scalable application.
Deploy an API to analyse images of headshots to identify an individual's generation cohort, supporting:
* Analysing images received via an API request.
* Providing access to a specified REST API, e.g. for use by front-end interfaces and other applications.
* Remaining responsive while analysing images.
* Reporting statistics about the analysed images.
